# PRD

Chrome Extension 용으로 개발된 이 프로젝트를 웹앱에서 사용할 수 있게 변경해야 합니다.

## 임베디드 에이전트(Embedded Agent) 위젯 개발

Extension 형태가 아니라 웹사이트 내부에 AI 채팅 위젯을 직접 구축할 때 WebMCP를 활용할 수 있어야 합니다. 
- 플랫폼 종속성 탈피: 특정 브라우저 확장 프로그램 없이도 사이트 내에 포함된 AI가 navigator.modelContext를 통해 해당 페이지의 도구들을 즉시 인식하고 실행합니다. 
- 예를 들어, 웹페이지의 왼쪽에는 입력폼이 있고 오른쪽에는 AI 채팅 위젯이 있습니다. 
  - AI 채팅 위젯이 도구를 발견하고 호출합니다. 

## 사용해야 할 기술:
- TypeScript
- 번들러: Vite
